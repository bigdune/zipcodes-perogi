ZIP Code Download - Readme for 5-digit ZIP Code Database Products

Thank you for purchasing our products - we strive to have the most correct data available anywhere. In this file you will find a list of the most commonly asked questions as well as where to go if you have further concerns.

A ZIP Code is a 5-digit code that identifies a specific geographic delivery area. ZIP Codes can represent an area within a state, an area that crosses state boundaries (unusual condition), or a single building or company that has a very high mail volume. ZIP is an acronym for Zone Improvement Plan. ZIP Codes can cross city boundaries and county boundaries.

Frequently Asked Questions

Microsoft Excel:
Q. Why do I get a message saying "File not loaded completely"?
A. This occurs when trying to load a "comma-separated" or CSV file into Excel - Excel supports a maximum of about 65,500 rows - while most of our products contain more than that. You may want to load the data using a different program like Access, SQL Server - or even Notepad!

Q. Where's the rest of the ZIP Codes? I only see half of them!
A. Because Excel has a limitation on the amount of information it can display per worksheet, we have split the ZIP Codes up (for most Editions) onto two worksheets. On the bottom-left corner of Excel you should see a "tab" that says "ZIPCodes1" or "ZIPCodes2" - simply click that and you'll see the rest.
